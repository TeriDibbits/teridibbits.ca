WagTag UI Design

Objective

The objective of this project was to create and design an app for a wearable. We used our design skills and UI knowledge to create a working app prototype with a seamless flow.

Concept

The concept is a app paired with a QR code scanner. This app allows the user to train, report and help find lost pets, and add pets using the scanner.
